# Contributing Guidelines

Thank you for your interest in contributing to this curated resource for lycopene vascular research!

## How to Suggest a New Resource

1. **Open an Issue**: Use the repository's issue tracker to suggest a new tool, dataset, or reference. Please provide:
   - A descriptive title.
   - A brief description of the resource.
   - A link to the resource (GitHub repository, dataset URL, etc.).
   - Any relevant tags or categories.

2. **Discuss**: The maintainers and community will review the suggestion. You may be asked for additional details.

3. **Pull Request (Optional)**: If you'd like to directly add the resource to the README or other documentation, feel free to open a pull request with the changes. Please follow the existing formatting style.

## Scope
We aim to include open‑source tools, publicly available datasets, and peer‑reviewed literature that are directly relevant to the study of lycopene’s effects on cardiovascular health, especially vascular function. Please keep suggestions on‑topic.

## Code of Conduct
Be respectful and constructive in all discussions. This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/0/code_of_conduct/).

We look forward to building a valuable resource together!
